# 测试计划

## 单元测试

数据模型校验、时间戳、重复和过期数据；每个指标的正常值、空值、常数序列、短序列和极端值；机会分边界；市场状态；策略注册；仓位计算。

## 属性测试

机会分始终0至100；输入不变输出不变；任何未来数据不得影响过去决策；候选数不超过配置；未知策略永远拒绝。

## 集成测试

模拟数据源到扫描排名的全链路；数据源故障导致NO_TRADE；相关性过滤；配置加载失败安全停止。

## 后续阶段测试

订单簿序列缺口和重同步；AI错误JSON；MT5断线、重启和重复信号；Demo账户判断；止损激活；经纪商拒单；状态恢复。

## 验收命令

项目应提供统一命令运行格式化检查、静态类型检查和全部离线测试。测试不得依赖网络或真实账户。
